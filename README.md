# Liv's Hus - Basic Tier

A luxury villa website for Liv's Hus in Apokoronas, Crete - Basic package featuring refined, editorial-style design inspired by premium villa rental platforms.

## ✅ Security Update

- **Next.js 15.1.9** - Patched against CVE-2025-55182 (Critical RCE vulnerability)
- Secure and production-ready

## Design Philosophy

Inspired by luxury villa platforms:
- Villas of Distinction
- Marriott Homes & Villas
- onefinestay
- Landfolk

### Key Features

- **Editorial Photography**: Hero-led visual storytelling
- **Refined Typography**: Playfair Display (serif) + Inter (sans-serif)
- **Luxury Color Palette**: Sand and ocean tones
- **Location-Focused**: Destination as hero element
- **Premium Positioning**: Curated, not listing-style

## Tech Stack

- Next.js 15.1.9 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons

## Installation

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm start
```
